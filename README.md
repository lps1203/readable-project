
# How to use this app

## Start the API server
  - Open a terminal
  - Make sure you're in the "readable-project" directory
  - type "cd api-server"
  - type "npm install"
  - type "node server" - this starts the backend server

## Start the app
  - Open another terminal
  - Make sure you're in the "readable-project" directory
  - type "cd frontend"
  - type "npm install"
  - type "npm start" - this starts the app
